# Bulk File Organizer (Python)

A simple Python tool to clean and organize messy folders in seconds.

## What it does
- Organizes files into folders by type (Images, Documents, Audio, Videos, Others)
- Optional bulk renaming with a custom prefix
- Works on Windows, macOS, and Linux
- No external libraries required

## Common use cases
- Messy Downloads folder
- College notes and PDFs
- Project files
- Media folders

## Usage
```bash
python organize.py --path <folder_path>
python organize.py --path <folder_path> --prefix MyFiles
```

### Example
```bash
python organize.py --path Downloads --prefix ProjectX
```
